# Demo Data

This directory contains example chat conversations for testing and demonstration purposes.

## Chat Files

- **safe_chat.txt**: A normal, healthy conversation between friends
- **bullying_example.txt**: Example of bullying behavior
- **manipulation_example.txt**: Example of manipulative language
- **grooming_example.txt**: Example of grooming patterns (age-inappropriate behavior)

## Usage

These demo chats can be selected from the sidebar in the ChatCompanion application to see how the detection system works.

## Note

These are fictional examples created for educational and demonstration purposes only.
